Prod (BallsForBrains) vs Shadow Models

1. Prod (Bayesian + SV)

BallsForBrains

--
The Incumbent Champion

This is the live production model executing trades with real money. It is a hybrid system utilizing PyMC3 Bayesian Stochastic Volatility to calculate deeply probabilistic market structures.

Unlike the shadows, it is paired with an aggressive Intraday VWAP Sniper that dynamically manages positions during the trading day, aborting bad trades and protecting capital using live market data.

2. Shadow Transformer

Deep Learning Attention

--
The Modern Challenger

This is a purely experimental Deep Learning model using a Multi-Head Attention Transformer (like an LLM for stocks).

It predicts sequential price action probabilities based on 60-day historical contexts. It trades strictly 'blind' during the day (buying at Open, selling at Close) and possesses absolutely zero intraday awareness.

3. Sandbox V1 Classic

Random Forest

--
The Legacy Baseline

This is the original AntiGravity engine before we upgraded to Bayesian math. It uses a standard Random Forest Regressor.

It is kept alive strictly as a benchmark to ensure that our newer, more complex models are actually outperforming the old baseline. It uses static, rigid stop-losses.

4. Shadow LSTM

Recurrent Network

--
The Sequential Predictor

This is an experimental Deep Learning model using a Long Short-Term Memory (LSTM) neural network.

It predicts time-series momentum based on historical sequences. Like the Transformer, it trades strictly 'blind' during the day with zero intraday awareness.
